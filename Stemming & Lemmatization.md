# Lemmatization

- Lemmatization is a word normalization technique that reduces a word to its dictionary base form,called the lemma.
- Unlike stemming, the output of lemmatization is always a real, meaningful word.

**Example**:

```
 “running” → run
 “better” → good
 “studies” → study
 “mice” → mouse
```

Lemmatization uses:

1. Vocabulary
2. Parts-of-Speech tagging
3. Morphological Analysis

**Example**:

```
bats(plural)--> bat(animal)
bats(verb)-->bat(verb)
```

Common Lemmatization Algorithms:
1. WordNet Lemmatizer
2. SpaCy Lemmatizer
3. Rule Based Morphological Lemmatizer
